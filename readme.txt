To run the Application:
    Clone this repo.
    Run the following commands:
        npm install
    and then,
        nodemon server.js

server.js is the main file (driver code)
    It contains, server conigurations and initialization

db.js contains db connection

static/
    contans all the static content such as HTMLs etc

joi-schema.js contains validation schema for validation
    Joi Reference: https://www.digitalocean.com/community/tutorials/how-to-use-joi-for-node-api-schema-validation
